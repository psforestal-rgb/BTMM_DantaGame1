# Hoja de ruta visual y de ampliación del juego

## Fase 1 — Identidad principal

- [x] Sistema visual modular.
- [x] Tema cute-comic-kawaii.
- [x] Mamá danta como personaje guía.
- [ ] Ajustar escala final tras prueba en móvil.
- [ ] Añadir expresiones adicionales: alegría, cansancio, miedo y protección.

## Fase 2 — Familia de dantas

- [ ] Rediseñar crías con manchas juveniles simplificadas.
- [ ] Variaciones de personalidad visibles.
- [ ] Animación de reunión con la madre.
- [ ] Celebraciones familiares coherentes.

## Fase 3 — Bosque y alimentación

- [ ] Plantas comestibles diferenciadas.
- [ ] Aguacatillos y frutos con estados de maduración.
- [ ] Arbustos de refugio.
- [ ] Agua y charcos.
- [ ] Partículas de hojas, semillas y lluvia.

## Fase 4 — Amenazas

- [ ] Vehículos por tipo con siluetas legibles.
- [ ] Luces y frenado más claros.
- [ ] Cazadores estilizados sin violencia gráfica.
- [ ] Trampas visibles y comprensibles.

## Fase 5 — Interfaz

- [ ] HUD simplificado para teléfono.
- [ ] Iconos propios en lugar de emojis inconsistentes.
- [ ] Tutorial ilustrado.
- [ ] Menús y resultados con escenas de la familia.

## Fase 6 — Nuevas mecánicas

- [ ] Eventos de educación ambiental.
- [ ] Señalización vial y reductores de velocidad.
- [ ] Restauración de pasos de fauna.
- [ ] Recolección de conocimiento sobre la especie.
- [ ] Álbum de escenas y fichas educativas.

## Criterio de aceptación por elemento

Cada elemento nuevo debe evaluarse en:

1. Identidad visual.
2. Lectura en pantalla pequeña.
3. Coherencia con Mamá Danta.
4. Rendimiento.
5. Accesibilidad.
6. Comprensión educativa.
