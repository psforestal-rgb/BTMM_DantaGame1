# Capa visual modular

Esta carpeta contiene la nueva capa de apariencia de BTMM DantaGame1.

- `css/theme-kawaii.css`: lenguaje visual general de interfaz.
- `js/visual-config.js`: configuración y banderas de activación.
- `js/visual-system.js`: registro y enrutamiento de renderizadores.
- `js/art/`: personajes, objetos y escenarios sustituidos por separado.

No deben modificarse colisiones ni reglas del juego desde estos archivos. La apariencia puede evolucionar de forma independiente del motor existente.
